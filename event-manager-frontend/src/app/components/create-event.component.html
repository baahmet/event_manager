Créer un événement

Le titre est requis. Le titre doit contenir au moins 3 caractères.
La description est requise. La description doit faire au moins 10 caractères.
La date de début est requise.
La date de fin est requise. La date de fin doit être postérieure à la date de début.
Le lieu est requis.
La capacité est requise. La capacité doit être au moins 1.
